Global Variable Syntax:

[] - Optional entities
<> - mandatory entities 

[access_specifier][access_modifier]<datatype><variablename> [= <literals>];

Access Specifier:
	public
	protected
	private
	*default*
Access Modifiers:
	static
	final
	transient
